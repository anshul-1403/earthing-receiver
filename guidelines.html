Smart Earth Monitor

Team StrawHats - Problem Statement ID: 1582

Idea/Solution

  • Leakage Current Calculation: Calculating current through changes in flux.
  • Earth Resistance Measurement: Measuring resistance using the fall of potential method (3-pole).
  • Street Light Monitoring: Checking if street lights are glowing for timely maintenance.
  • Self-error detection: Automatically triggers alerts for malfunctions.
  • Data Transmission: Transmitting data over a wide range without internet.
  • Storing Data: Using an SQL database for future use.
  • Monitoring: Displaying the data on a web application.

Problem Resolution

  • Micro-controller: Arduino compatibility with sensors.
  • Real-time Monitoring: Instant alerts for earth leakage detection.
  • Remote Deployment: Easy setup in remote locations.

Uniqueness

  • Non-Contact Measurement: Accurate current measurement.
  • Customisable: Modular solution for specific needs.

Impact and Benefits

  • Enhanced Electrical Safety.
  • Proactive Fault Detection.
  • Reduced Equipment Downtime.
  • Improved Productivity.
  • User-Friendly Interface.
  • Customizable Alerts.
